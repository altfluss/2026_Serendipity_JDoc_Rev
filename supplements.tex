\section{Experimental Data Preparation}\label{sup:data_prep}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

We carefully prepared input features to be as unfolded as possible
before OT-FB learning. Tags explicitly naming genres were excluded,
and descriptors were composed into meaningful sentences rather than
bare lists. Even so, descriptors still carry implicit genre
associations; full disentanglement of genre-ness and atmospheric structure
is left to the OT-FB learning.

\paragraph{Music dataset.}
From the Million Song Dataset~\cite{Bertin-Mahieux2011}, we selected 2,500 songs
evenly distributed across ten genres: rock, pop, jazz, classical, electronic,
hip-hop, folk, country, metal, and blues. Each song carries a single
hard-assigned genre label. We extracted 237 high-frequency descriptive tags from
Last.fm annotations, explicitly excluding genre labels. Tags were formatted as
``These are vibe-only tags describing the mood and style of a music track:
\textit{tag\textsubscript{1}}, \ldots, and \textit{tag\textsubscript{N}}''
(median six tags per song) and embedded using \texttt{text-embedding-3-large}
(3072~dimensions, L2-normalized), then reduced to $d=10$ via PCA. A small TF-IDF
perturbation (weight ratio 0.1) was added to break ties between songs sharing
identical tag sets; this does not essentially affect the feature distribution.

\paragraph{Children's literature dataset.}
We curated 409 canonical children's books spanning fifteen genres including
fantasy, realistic fiction, adventure, historical fiction, mystery, and school
stories. Weighted (soft) genre assignments---up to three per book---were
estimated using GPT-4o (e.g., ``fantasy~(0.6), adventure~(0.3), school
story~(0.1)''). Because all selected books are well-known titles, we used GPT-4o
to automatically generate 153 semantic descriptors per book---adjectives
capturing tonal impressions (e.g., \textit{whimsical}, \textit{heartwarming})
and nouns capturing themes (e.g., \textit{friendship}, \textit{coming-of-age}).
Descriptors were formatted as ``This is a children's book. The impressions are
\textit{adj\textsubscript{1}}, \ldots\ The themes are
\textit{noun\textsubscript{1}}, \ldots'' (median 13 descriptors per book),
embedded using \texttt{text-embedding-3-small} (1536~dimensions, L2-normalized),
with the embedding, and reduced to $d=10$. A small TF-IDF perturbation (weight
ratio 0.1) was also added.

\section{Quantitative Comparison with Baselines}\label{sup:baselines}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

We compare the proposed method with three autoencoder-based disentanglement
baselines: $\beta$-VAE~\cite{Higgins2017}, $\beta$-TCVAE~\cite{Chen2018},
and WAE~\cite{Tolstikhin2018}. All models use a 4-dimensional latent space
$(u,v)\in\mathbb{R}^2\times\mathbb{R}^2$ and aim to separate category
information ($u$) from atmospheric structure ($v$).
Performance is evaluated using two complementary metrics.

The \textbf{Mutual Information Gap (MIG)}~\cite{Chen2018} measures
statistical independence between the two axes:
\[
    \text{MIG} = \frac{I[C;U]-I[C;V]}{H[C]},
\]
where $I[\cdot;\cdot]$ denotes mutual information and $H[\cdot]$ denotes
entropy. A higher MIG (approaching~1) means genre-ness and atmosphere are
more nearly independent. As noted in the main text, however, statistical
independence alone admits infinitely many solutions; it does not pin down
which of them aligns the categories.

The \textbf{OT Score} measures exactly that alignment---whether items
at homologous positions across categories are mapped to one another:
\[
    \text{OT Score} =
    \frac{D^2_{\text{random}}-D^2_V}
    {D^2_{\text{random}}-D^2_{\text{OT}}},
\]
where $D^2_{\text{OT}}$ is the optimal transport distance in the original
feature space~$X$, $D^2_V$ uses the learned atmosphere coordinates~$V$ to
compute the transport plan applied to~$X$, and $D^2_{\text{random}}$
provides a random transport baseline. An OT Score approaching~1 indicates
that this cross-category correspondence is faithfully preserved in the
learned latent space.

Table~\ref{tab:comparison_results} shows results averaged over 10 trials.
All methods achieve comparable MIG scores: statistical independence does
not single out one approach. What distinguishes them is the OT Score---
whether that independence is reached while preserving cross-category
correspondence. Here the proposed method consistently scores higher on
both datasets, with standard deviations an order of magnitude smaller,
suggesting a more stable alignment. Among the baselines, WAE comes
closest, consistent with its use of optimal transport in training.

\section{Soft-Category Extension on Children's Literature}\label{sup:children}

The OT-FB framework assumes each item belongs to a single category.
For children's literature, where books often span multiple genres,
we extend the framework via soft memberships.

\paragraph{Two-step approach.}
When explicit categories are unavailable, we infer latent category
coordinates from auxiliary features (e.g., metadata, tags).
First, given auxiliary feature vectors $y_i$, we infer
$u_i \in \mathcal{U}$ by assuming $y_i = \psi(u_i)$ and minimizing:
\[
    E[U] = \frac{1}{N} \sum_i \left\{
    \frac{\beta}{2} \| y_i - \psi(u_i) \|^2 - \log q(u_i)
    \right\}.
\]
Second, we define soft memberships via local neighborhoods:
\[
    \rho(j \mid i) =
    \frac{\mathcal{N}(u_j \mid u_i, \lambda^{-1} I)}
    {\sum_{j'} \mathcal{N}(u_{j'} \mid u_i, \lambda^{-1} I)}.
\]
These soft memberships replace hard category assignments,
accommodating overlapping or fuzzy genre boundaries.
Once $u_i$ are estimated, atmosphere coordinates $v_i$ are extracted
through the same OT-FB learning as the music dataset.

\paragraph{Results.}
Figure~\ref{fig:children_bundle} shows the learned fiber bundle structure.
Figure~\ref{fig:children_uv}(a) and~(b) show the learned structure of
$\mathcal{U}$ and $\mathcal{V}$, respectively. In $\mathcal{U}$, genres are
arranged continuously along two readable contrasts: a cold--warm axis,
with mystery and horror at one end and picture books and poetry/wordplay
at the other, and a fictional--realistic axis, with fantasy and myths on
one side and drama on the other. In $\mathcal{V}$, the atmosphere is
organized by two contrasts as well: a light--heavy axis, from playful and
humorous to tragic and melancholic, and a tense--restful axis, from
thrilling and dramatic to warm and pastoral. Together these two contrasts
form a cycle, atmospheric character progressing continuously across genre
boundaries---the same structure observed in the music dataset. The two
axes $\mathcal{U}$ and $\mathcal{V}$ are independent: one can move across
genres while staying at the same atmosphere coordinate, or vary genre-ness
while remaining within an atmosphere region. The independence is confirmed
quantitatively by the MIG and OT Scores reported in
Table~\ref{tab:comparison_results}.

A single item may be embedded through more than one view. As noted in
the main text, our children's literature instantiation does exactly
this: from the same books, the noun tags and the adjective tags give
two descriptions, hence two body data, each carving out its own
universe with its own rabbit hole.

\paragraph{Two body data.}
For each book we form two texts. The noun tags become
``This is a children's book. The themes are
\textit{noun\textsubscript{1}}, \ldots,'' and the adjective tags become
``This is a children's book. The impressions are
\textit{adjective\textsubscript{1}}, \ldots.'' Each text is embedded and
learned through the same OT-FB procedure as in
Section~\ref{sup:children}, over a shared genre-ness space $\mathcal{U}$.
The single atmosphere space $\mathcal{V}$ of Section~\ref{sup:children}
thus splits into two: the noun-tag body yields a \textbf{thematic
atmosphere} $\mathcal{V}_{\mathrm{thematic}}$, and the adjective-tag body
yields a \textbf{tonal atmosphere} $\mathcal{V}_{\mathrm{tonal}}$.

\paragraph{What appears.}
Figure~\ref{fig:twobody_spaces} shows the three spaces. The genre-ness
space~(a) reproduces the structure of Figure~\ref{fig:children_uv}(a),
as theory predicts---the two body data share the same $\mathcal{U}$. The
two atmosphere spaces differ. The thematic atmosphere
$\mathcal{V}_{\mathrm{thematic}}$~(b) recovers the literature counterpart
of the affective structure seen in music: cold themes such as truth,
oppression, war, and isolation gather at one pole, warm themes such as
belonging, kindness, and empathy at the other, while an exciting direction
(curiosity, problem-solving, discovery) runs against a calm one (hope,
resilience). The tonal atmosphere $\mathcal{V}_{\mathrm{tonal}}$~(c)
appears less cyclic and closer to a one-dimensional spectrum, running from
playful and cheerful at one end to tragic and grim at the other.

\section{Complete Questionnaire (20~Items)}\label{sup:questionnaire}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Items 3--14 and 16--18 are adapted from the \textit{Serendipitous Digital
    Environment Scale}~\cite{McCayPeet20181}, a validated instrument for
evaluating digital information environments. We replaced the original
placeholder ``[the digital information environment]'' with ``this system''
and adapted general terms (e.g., ``information'') to the music discovery
context (e.g., ``content (music)''). Items 1, 2, 15, 19, and 20 were
developed for this study.

\paragraph{Category A: Ease of discovery (developed for this study)}
\begin{enumerate}
    \item I found it easy to discover my preferred content (music) with this system.
    \item I was able to find my preferred content (music) with this system.
\end{enumerate}

\paragraph{Category B: Leads to the unexpected (McCay-Peet \& Toms)}
\begin{enumerate}
    \setcounter{enumi}{2}
    \item I bumped into unexpected content (music) with this system.
    \item I came across content (music) by chance with this system.
    \item I was exposed to unanticipated content (music) with this system.
    \item I stumbled upon content (music) unexpectedly with this system.
    \item I encountered unforeseen content (music) with this system.
\end{enumerate}

\paragraph{Category C: Trigger-rich (McCay-Peet \& Toms)}
\begin{enumerate}
    \setcounter{enumi}{7}
    \item This system was full of information useful to me.
    \item I found information of value to me with this system.
    \item This system was a treasure trove of information.
\end{enumerate}

\paragraph{Category D: Highlights triggers (McCay-Peet \& Toms + original)}
\begin{enumerate}
    \setcounter{enumi}{10}
    \item This system had features that drew my attention to information.
    \item This system had features that alerted me to information.
    \item This system had features that ensured my attention was drawn to useful information.
    \item This system had mechanisms that guided me from one piece of content (music) to another.
    \item This system allowed me to explore content beyond existing classifications.\textsuperscript{\dag}
\end{enumerate}
\textsuperscript{\dag}Originally developed for this study.

\paragraph{Category E: Enables connections (McCay-Peet \& Toms)}
\begin{enumerate}
    \setcounter{enumi}{15}
    \item I was able to see connections between pieces of content (music) within this system.
    \item This system helped me connect different pieces of content (music).
    \item This system enabled me to understand relationships among content (music).
\end{enumerate}

\paragraph{Category F: Discovering preferences (developed for this study)}
\begin{enumerate}
    \setcounter{enumi}{18}
    \item I discovered new interests or made new discoveries by using this system.
    \item This system helped me gain a deeper understanding of my own preferences and tastes.
\end{enumerate}

report entropy differences for Experiments~1 and~2.
Significance levels throughout: {*}~$p < 0.05$, {**}~$p < 0.01$,
{***}~$p < 0.001$.

\paragraph{Experiment 1 (P vs.\ C1).}
Table~\ref{tab:behavior_change_exp1} shows entropy differences between
systems P and C1.
In Group CP, genre breadth in discovery increased when using system~P
($+0.115$, $p < 0.01$); in Group PC, the difference was not significant

\paragraph{Experiment 2 (P vs.\ C2).}

sequence (C2b$\to$P$\to$C2a).

$p < 0.001$) and remained elevated in C2a
(P$\to$C2a: $+0.268$ to $+0.275$, $p < 0.001$).
Atmosphere exploration narrowed in~P ($-0.040$, $p < 0.001$) and showed
no significant further change in C2a ($-0.007$, $p = 0.197$).
